# LuckPacket

> 微信快速领取最大红包

## 机器人微信
<img src="https://raw.githubusercontent.com/Lrony/LuckPacket/master/QR.jpg" width="200px" />

## 使用说明
- 添加机器人为好友
- 转发外卖红包给机器人
- 输入需要领取红包的手机号
- End

## 开发
- 运行环境为`Python3.4`
- 运行`robot.py`后扫码登陆
